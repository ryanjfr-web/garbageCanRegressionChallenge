4. Key Insights for Your Analysis

For Question 2 (StressSurvey vs Anxiety):
• The scatter plot shows a strong positive linear relationship
• R² = 0.9011 indicates very good fit
• However, the coefficient (1.047) is close but not exactly 1.0
• This demonstrates that even with good fit, omitted variables cause bias

For Question 4 (Time vs Anxiety):
• The scatter plot shows a positive relationship but poor fit
• R² = 0.5630 indicates moderate fit
• The coefficient (5.3406) is dramatically wrong compared to true value (0.1)
• This shows severe omitted variable bias when using Time alone

Overall Lesson:
• High R² doesn't guarantee correct coefficients
• Omitted variable bias can be severe
• Need multiple regression with all relevant variables for correct estimates
